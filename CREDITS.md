# Credits

This application is made possible by the following open-source projects and web services:

## Data & Web Services

### [OpenNorth](https://opennorth.ca/)
Special thanks to OpenNorth for providing the [Represent API](https://represent.opennorth.ca/), which serves as the primary data source for legislative members, ridings, and party affiliations.

### Official Legislature Websites
Biographical information and official portrait images are sourced via the OpenNorth API from the federal and provincial legislative websites.

## Core Technologies

### [Flutter](https://flutter.dev/)
An open-source UI software development kit created by Google for building natively compiled applications for mobile, web, and desktop from a single codebase.

### [Drift (formerly Moor)](https://drift.simonbinder.eu/)
A reactive persistence library for Flutter and Dart, built on top of SQLite, used for profile management and local data caching.

### [FSRS Algorithm](https://github.com/open-spaced-repetition/fsrs4dart)
Uses the Free Spaced Repetition Scheduler (FSRS) algorithm to optimize learning intervals and improve memorization efficiency.

## Third-Party Packages

- **[Dio](https://pub.dev/packages/dio)**: A powerful HTTP client for Dart, used for API interaction.
- **[Cached Network Image](https://pub.dev/packages/cached_network_image)**: For efficient on-demand image loading and caching.
- **[Provider](https://pub.dev/packages/provider)**: For state management and dependency injection.
- **[Google Fonts](https://fonts.google.com/)**: For beautiful, modern typography.
- **[Path Provider](https://pub.dev/packages/path_provider)**: For finding commonly used locations on the filesystem.
- **[Sqlite3](https://pub.dev/packages/sqlite3)**: The underlying database engine for robust local storage.
