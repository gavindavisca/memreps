# Credits

This application is made possible by the following open-source projects, web services, and contributors:

## Data & Web Services

### [OpenNorth](https://opennorth.ca/)
Special thanks to OpenNorth for providing the [Represent API](https://represent.opennorth.ca/), which serves as the primary data source for legislative members, ridings, and party affiliations across Canada.

### Official Legislature Websites
Biographical information and official portrait images are sourced via the OpenNorth API from the House of Commons and provincial legislative websites.

### [Firebase](https://firebase.google.com/)
Provides the backend infrastructure for user profile synchronization, leaderboard calculations, and image proxy services via Cloud Functions and Firestore.

## Core Technologies

### [Flutter](https://flutter.dev/)
An open-source UI software development kit created by Google for building natively compiled applications for mobile, web, and desktop from a single codebase.

### [Drift](https://drift.simonbinder.eu/)
A reactive persistence library for Flutter and Dart, built on top of SQLite, used for robust local profile management and data caching.

### [FSRS Algorithm](https://github.com/open-spaced-repetition/fsrs4dart)
Uses the Free Spaced Repetition Scheduler (FSRS) algorithm to optimize learning intervals and improve memorization efficiency through scientific spaced repetition.

## Third-Party Packages

- **[fl_chart](https://pub.dev/packages/fl_chart)**: Powering the beautiful progress visualizations and rolling average charts on the Results page.
- **[Provider](https://pub.dev/packages/provider)**: For reliable state management and dependency injection across the application.
- **[Dio](https://pub.dev/packages/dio)** & **[http](https://pub.dev/packages/http)**: For efficient API interactions and backend communication.
- **[Cached Network Image](https://pub.dev/packages/cached_network_image)**: For high-performance on-demand image loading and caching.
- **[Google Fonts](https://fonts.google.com/)**: For the clean, modern typography (Outfit and Inter) used throughout the UI.
